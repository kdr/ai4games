Getting Started with Luigi Bros Game

Welcome to the Luigi Bros platform game! This guide will help you set up and start playing the game.

Setup Instructions

  1. Generate Game Assets

    Before you can play the game, you need to create the image and sound assets:

    Open the Image Asset Generator and generate all image assets by clicking the "Generate" button for each asset and then "Save" to download them.

    Then open the Sound Asset Generator and do the same for all sound effects and music.

  2. Organize Assets

    Place all generated assets in the correct folders:

    • Put all image files (.png) in the assets/images/ folder
    • Put all sound files (.wav, .mp3) in the assets/sounds/ folder

    You can use the Asset Manager to help organize your files.

  3. Play the Game

    Once all assets are in place, open index.html in your web browser to start playing!

    Controls:

    • Arrow Left/Right: Move Luigi
    • Arrow Up: Jump
    • Space: Start game (on title screen)

Game Objective

As Luigi, your goals are:

  • Collect all coins to win the game
  • Jump on enemies to defeat them
  • Avoid being hit by enemies from the sides
  • Reach the highest score possible

Troubleshooting

If the game doesn't load or shows errors:

  • Make sure all assets are generated and placed in the correct folders
  • Check browser console for errors (F12 in most browsers)
  • Try using a modern browser like Chrome, Firefox, or Edge
  • If files appear to be missing, verify the file names match exactly what's expected
